The all-seeing read gateway.

For developers

Call Pythia without embedding node routing or Pact decoding.

npm install @ancientpantheon/pythia-client
  • GET /healthzservice + node-pool status
  • GET /api/v1/getBalance?chain=stoachain&address=…composite decoded balance
  • GET /api/v1/getConfirmations?chain=stoachain&tx=…pending vs final + depth
  • POST /stoachain/rpcverbatim relay to a healthy node
