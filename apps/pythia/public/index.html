Pythia

The all-seeing gateway.

One endpoint for pooled, failover-safe blockchain transport — read any chain, broadcast your own signed transactions, and poll their status. Consumers ask Pythia; Pythia handles the nodes, the routing, and the failover. Keyless — it never holds keys and never signs; it only relays what you give it.

For developers

Call Pythia without embedding node routing — reads, broadcasts, and confirmations behind one base URL.

npm install @ancientpantheon/pythia-client
  • GET /healthzservice + node-pool status
  • POST /stoachain/readdirty read — caller supplies Pact code
  • POST /stoachain/sendkeyless broadcast — relay caller-signed txs
  • POST /stoachain/polltx status — pending vs final + depth
